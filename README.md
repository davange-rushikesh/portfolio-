# Antigravity Portfolio

Premium futuristic developer portfolio built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Edit your content

Update `data/profile.ts` to replace placeholders:

- `[YOUR_NAME]`
- `[YOUR_ROLE]`
- `[YOUR_BIO]`
- `[YOUR_EMAIL]`
- `[YOUR_PHONE]`
- `[YOUR_LOCATION]`
- `[YOUR_GITHUB]`
- `[YOUR_LINKEDIN]`
- `[YOUR_INSTAGRAM]`
- `[YOUR_PROFILE_IMAGE]`
- `[YOUR_RESUME_LINK]`
- project details in `profile.projects`

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`

## Notes

The site is structured for fast page loads, clean components, and responsive dark UI with glassmorphism and soft glow effects.
